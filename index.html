You can also slow down the song for practicing tough sections.
Note that this is a really shitty, basically useless feature. It will affect the time between notes, not the speed of the individual notes. It also doesn't change the music itself, so it will be offbeat, and the song will end before the notes do. This means you'll only be able to practice the first half of the notes if you use the 2x slower option, and so on. If you really want to practice all the notes on a slower speed, try replacing the song file with a longer song.
Seriously, think twice before changing this.
